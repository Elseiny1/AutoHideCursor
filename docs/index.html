AutoHide Cursor

AutoHide Cursor is a lightweight Windows tray app that automatically hides your mouse cursor after a short period of inactivity. Set a timeout that fits your workflow, quickly toggle the behavior from the tray, change the tray icon, and choose whether it runs at startup. Runs as a standard user – no admin required.

Key Features

Automatic Cursor Hiding

After your chosen idle time, the mouse cursor fades away so it never distracts you.

Configurable Timeout

Quickly cycle between presets directly from the tray menu.

Theme-Aware Tray Icon

Uses a light or dark tray icon depending on your taskbar theme.

Run at Startup

Enable or disable auto-start with a single menu option.

Persistent Settings

Your timeout, enabled state, and custom icon are remembered between sessions.

Auto-Update

On launch, the app checks for a newer release and will update itself.
